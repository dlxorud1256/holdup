# 🃏 홀덤 연습장 (holdup)

초보자를 위한 텍사스 홀덤 연습 게임입니다. AI 봇(1~5명 선택)과 함께 플레이하면서 홀덤의 규칙과 감을 익힐 수 있어요. 헤즈업(1:1)부터 6인 풀 테이블까지.

## 초보자 배려 기능

- **📖 족보표** — 카드 예시와 함께 족보 순위를 언제든 확인
- **❓ 게임 방법** — 첫 접속 시 자동으로 열리는 규칙 설명
- **현재 족보 표시** — 내 패가 지금 어떤 족보인지 실시간 표시, 최고 조합 5장은 금색 테두리로 하이라이트
- **실시간 승률 표시** — 몬테카를로 시뮬레이션으로 계산한 내 패의 승률을 항상 표시
- **💡 힌트 버튼** — 승률과 팟 오즈를 비교한 수학적 근거와 함께 추천 행동을 한글로 설명
- **🎓 AI 코치 (선택)** — OpenAI API(GPT-5.4 mini)를 연결하면 "왜 콜이 좋아?",
  "팟 오즈가 뭐야?" 같은 자유 질문을 실시간 채팅으로 물어볼 수 있음. 현재 게임
  상황(내 카드, 승률, 추천)을 자동으로 참고해 답변. OpenAI API 키 필요
  (브라우저에만 저장, 봇 카드는 전송 안 함). OpenAI는 브라우저 직접 호출을
  차단하므로 `proxy/`의 Vercel 함수(미국 리전)가 키 저장 없이 중계함
- **📋 자동 핸드 복기** — 코치 연결 시 핸드가 끝날 때마다 잘한 점·아쉬운 점을 짧게 복기.
  참여하지 않은 핸드(프리플랍 즉시 폴드)는 생략해서 API 비용 절약
- **🫣 훔쳐보기** — 핸드 종료 후 버튼을 누르면 숨겨진 상대 패 공개 + 남은 보드까지
  깔아서 "끝까지 갔다면 누가 이겼나" 확인(래빗 헌팅). 실전에 없는 연습용 특권이라
  봇들의 관찰 통계에는 반영되지 않음
- **🕘 핸드 히스토리** — 끝난 핸드들을 판 단위로 기록(최근 50개). 내 카드·보드·전체
  액션·순손익을 다시 보고, 버튼 하나로 그 핸드를 코치에게 물어볼 수 있음
- **한글 액션 버튼** — 폴드/체크/콜/레이즈 버튼마다 짧은 설명 부착
- **게임 기록** — 모든 진행 상황이 한글 로그로 남음

## 게임 규칙 구현

- 텍사스 홀덤 풀 룰: 프리플랍 → 플랍 → 턴 → 리버 → 쇼다운
- **두 가지 모드**: 토너먼트(블라인드 6핸드마다 인상, 전원 탈락시키면 우승) /
  캐시 게임(블라인드 고정, 파산한 봇 자리에 새 성향의 손님 착석, 리바이 가능)
- 딜러 버튼 로테이션, 헤즈업 블라인드 규칙
- 올인 & 사이드팟 정산, 무승부 팟 분할
- 포지션 인식 봇(버튼에서 넓게, 이른 위치에서 타이트하게), 멀티웨이 블러핑 감소,
  쇼다운에서 들킨 블러프 기억, 결정 난이도에 따른 타이밍 텔
- **몬테카를로 시뮬레이션 AI** — 봇이 매 결정마다 수백 번의 가상 쇼다운을 돌려
  실제 승률을 추정하고, 그 위에 봇별로 다른 전략 층을 얹음
- **비밀 아키타입 시스템** — 매니악·콜링 스테이션·함정형·바위·균형형 5가지 유형 풀에서
  새 게임마다 3개를 비밀리에 뽑아 봇들에게 배정. 누가 어떤 타입인지는 직접 관찰해서
  알아내야 하고, 같은 유형이라도 게임마다 강도가 다름
- **오프타입 믹싱** — 핸드의 약 12%는 자기 유형과 다르게 쳐서 읽기가 확신이 되지 않게 함
- **적응형 관찰** — 봇들도 사람의 폴드율·참여율을 추적해 블러핑/밸류 베팅 빈도를 조절.
  코치는 봇별 통계(참여율·레이즈 비율)를 단서로 유형 추리를 함께 도와줌

## 실행 방법

```bash
npm install
npm run dev
```

브라우저에서 http://localhost:5173 접속.

## 기술 스택

React 18 + TypeScript + Vite. 서버 없이 브라우저에서만 동작합니다.
