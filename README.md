# 🃏 홀덤 연습장 (holdup)

초보자를 위한 텍사스 홀덤 연습 게임입니다. AI 봇 3명과 함께 플레이하면서 홀덤의 규칙과 감을 익힐 수 있어요.

## 초보자 배려 기능

- **📖 족보표** — 카드 예시와 함께 족보 순위를 언제든 확인
- **❓ 게임 방법** — 첫 접속 시 자동으로 열리는 규칙 설명
- **현재 족보 표시** — 내 패가 지금 어떤 족보인지 실시간 표시, 최고 조합 5장은 금색 테두리로 하이라이트
- **실시간 승률 표시** — 몬테카를로 시뮬레이션으로 계산한 내 패의 승률을 항상 표시
- **💡 힌트 버튼** — 승률과 팟 오즈를 비교한 수학적 근거와 함께 추천 행동을 한글로 설명
- **🎓 AI 코치 (선택)** — Claude API를 연결하면 "왜 콜이 좋아?", "팟 오즈가 뭐야?" 같은
  자유 질문을 실시간 채팅으로 물어볼 수 있음. 현재 게임 상황(내 카드, 승률, 추천)을
  자동으로 참고해 답변. Anthropic API 키 필요(브라우저에만 저장, 봇 카드는 전송 안 함)
- **한글 액션 버튼** — 폴드/체크/콜/레이즈 버튼마다 짧은 설명 부착
- **게임 기록** — 모든 진행 상황이 한글 로그로 남음

## 게임 규칙 구현

- 텍사스 홀덤 풀 룰: 프리플랍 → 플랍 → 턴 → 리버 → 쇼다운
- 블라인드(50/100), 딜러 버튼 로테이션, 헤즈업 블라인드 규칙
- 올인 & 사이드팟 정산, 무승부 팟 분할
- **몬테카를로 시뮬레이션 AI** — 봇이 매 결정마다 수백 번의 가상 쇼다운을 돌려
  실제 승률을 추정하고, 그 위에 봇별로 다른 전략 층을 얹음
- **비밀 아키타입 시스템** — 매니악·콜링 스테이션·함정형·바위·균형형 5가지 유형 풀에서
  새 게임마다 3개를 비밀리에 뽑아 봇들에게 배정. 누가 어떤 타입인지는 직접 관찰해서
  알아내야 하고, 같은 유형이라도 게임마다 강도가 다름
- **오프타입 믹싱** — 핸드의 약 12%는 자기 유형과 다르게 쳐서 읽기가 확신이 되지 않게 함
- **적응형 관찰** — 봇들도 사람의 폴드율·참여율을 추적해 블러핑/밸류 베팅 빈도를 조절.
  코치는 봇별 통계(참여율·레이즈 비율)를 단서로 유형 추리를 함께 도와줌

## 실행 방법

```bash
npm install
npm run dev
```

브라우저에서 http://localhost:5173 접속.

## 기술 스택

React 18 + TypeScript + Vite. 서버 없이 브라우저에서만 동작합니다.
